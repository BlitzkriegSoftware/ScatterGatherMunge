# ScatterGatherMunge

The code accompanying the YouTube video of the same name

> Requires Docker

## Start Docker Images

```powershell
.\scripts\start.ps1
```
## Configuration

`demoConfig.json` is the default, but you can modify it or copy it and feed the EXEs your own. Same for the `scripts/`

It is an hash table of name/value pairs

```json
{
  "": "",
  "": ""
}
```
